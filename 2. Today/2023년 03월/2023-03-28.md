## CSS 최신 기술
---
[요즘 CSS 미쳤습니다...! 2023 최신 기능 쌉정리 - YouTube](https://www.youtube.com/watch?v=HZIcTZABMuc)

@supports 기술이 뜨고 있음.

@supports rule 은 조건부 CSS 코드를 쓸 수 있게 해줌.

```css
@support (text-align:center) {
	body {
		text-align:center
	}
}
```
