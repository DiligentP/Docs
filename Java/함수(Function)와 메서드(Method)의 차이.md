#Java

## 함수(Function)
	함수는 여러 문장들이 하나의 기능을 구현하도록 구성한 것이라고 할 수 있다.

```java
public void function() {
	...
	return result;
}
```

## 메서드(Method)
	메서드는 클래스 내부에 정의된 함수를 말한다.

```java
public class Class() {
	...
	public void method() {
		...
		return result;
	}
}
```

즉 둘을 구별하는 가장 큰 차이점은, 독립적으로 존재하냐, 클래스 내부에 종속되어있냐의 차이이겠다.
