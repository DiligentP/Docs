- `divmod()` [divmod() 내장 함수로 나눗셈하기 | 달레의 파이썬 프로그래밍 - YouTube](https://www.youtube.com/watch?v=Q_rBHHXl4Gg)

divmod() 함수를 이용하면, 나누기, 나머지 값을 모두 구할 수 있다.

```python
# 나누기 연산자 / (실수부분 포함)  // (몫만)
print(8 / 2) # 4.0
print(8 / 3)  #2.6666666666665
print(8 // 2) # 4
print(8 // 3) # 2

# 나머지 연산자 %
print(8 % 2) # 0
print(8 % 3) # 2

# divmod()
divmod(8, 2)  # (8//2,8%2) 와 같음.    # (4, 0)
```
