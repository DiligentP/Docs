

1. 주어진 Script를 실행하는 컨테이너를 빌드하시오. 
	컨테이너 이름 : fortune:20.02
	dockerfiel 내용 :
		base images: debian
		컨테이너에 아래의 webpage.sh 파일을 복사
		컨테이너에 fortune 애플리케이션 설치 : apt-get install fortune
		컨테이너 실행 시 저장한 webpage.sh가 실행되도록 구성

webpage.sh
```
#!/bin/bash
mkdir /htdocs
while :
do
	/usr/games/forune > /htdocs/index.html
	sleep 10
done
```
