
명령어
```
# 컨테이너 찾기
$ docker search <컨테이너>  #  hub.docker.com 에  컨테이너 이미지가 있는지 확인하고 알려줍니다.
```

사용 예
```
docker search nginx

...

[root@node1 ~]# docker search nginx
NAME               DESCRIPTION                     STARS     OFFICIAL   AUTOMATED
nginx              Official build of Nginx.        18400      [OK]       
unit               Official build of NGINX           0        [OK]       
bitnami/nginx      Bitnami nginx Docker Image       158       [OK]
...

```
