
# Windows & Linux
---
`Ctrl + T` : 새 탭 열기
`Ctrl + Tab` :  다음탭으로 이동

`Ctrl + Shift + A`  열려 있는 탭 검색


`F5` : 새로고침
`Shift + F5` : 웹페이지 렌더링  + 새로고침

`Ctrl + K`  : 구글검색 바로 이동
`Ctrl + Shift + B`  : 북마크 ON/OFF
